# Assistant (Roblox Studio)

## Overview
Assistant is an AI-powered helper built into Roblox Studio, designed to accelerate content creation, automate repetitive tasks, and provide scripting and design guidance directly within the editor.

### Key Links
- [Assistant Guide](https://create.roblox.com/docs/assistant/guide)
- [RDC 2024 AI Announcements](https://braintitan.medium.com/roblox-unveils-ai-driven-3d-tools-for-real-time-world-building-ac693bd3d641)

## Capabilities
- Generate and edit scripts in place
- Add, modify, and bulk-update objects in the data model
- Insert assets from the Creator Store
- Explain code and answer development questions
- Generate materials and 3D models (beta)

### Key Links
- [Prompt Guide](https://create.roblox.com/docs/assistant/guide)

## Pros & Cons
**Pros:**
- Greatly accelerates prototyping and iteration
- Reduces manual scripting and object placement
- Integrated directly in Studio
- Continuously updated with new features

**Cons:**
- Generated code may require review and adjustment
- Some features (e.g., 3D model generation) are in beta

### Key Links
- [Assistant Overview](https://create.roblox.com/docs/assistant/guide)

## Setup Instructions
1. Open Roblox Studio and click the Assistant button in the toolbar.
2. Type requests or questions in the Assistant chat window.
3. Use prompt guide for best results.
4. Enable beta features as needed (e.g., mesh generation).

- [Assistant Setup](https://create.roblox.com/docs/assistant/guide)
